## ðŸ“ Common Actions

### Create New Item

Click the **+** button in any module list.

### Edit Item

Click on any row in a table, or click the edit icon.

### Delete Item

Click the delete icon on any row. You'll be asked to confirm.

### Duplicate Item

Click the copy icon to create a copy with "(Copy)" suffix.
